Routing
    open index.html and add <base href="/">
    create a file under app folder name: app-routing.module.ts
        type ng generate module app-routing --flat --module=app
        import RouterModule and Routes and add it to exports
    configure the routes on app-routing.module.ts
    export all the components so that app.module can use it
    import the configured routes by typing this [ RouterModule.forRoot(routes) ] under the imports array
    add <router-outlet> tag on the app.component.html
